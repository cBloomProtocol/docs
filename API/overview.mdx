Bloom Protocol Open API allows builders to programmatically manage their Social Missions, view submissions, and approve rewards.

## Base URL

```
https://connect.bloomprotocol.ai
```

## API Documentation

Interactive API documentation is available at:

```
https://connect.bloomprotocol.ai/docs
```

## What You Can Do

<CardGroup cols={2}>
  <Card title="Query Missions" icon="list">
    Get list of active Social Missions and their details
  </Card>
  <Card title="View Submissions" icon="inbox">
    See all submissions for your missions with AI review scores
  </Card>
  <Card title="Approve/Reject" icon="check">
    Programmatically approve or reject submissions
  </Card>
  <Card title="Usage Analytics" icon="chart-line">
    Track your API usage and statistics
  </Card>
</CardGroup>

## Two Tiers of Access

| Tier | Authentication | Access |
|------|---------------|--------|
| **Public** | None (IP rate limited) | View missions and public submission data |
| **Builder** | API Key required | Full access including AI review scores, approve/reject actions |

## Quick Start

<Steps>
  <Step title="Get your API Key">
    Go to [Builder Dashboard](https://preview.bloomprotocol.ai/builder/api-keys) and create an API key
  </Step>
  <Step title="Make your first request">
    ```bash
    curl -X GET "https://connect.bloomprotocol.ai/open/v1/missions" \
      -H "x-api-key: your_api_key_here"
    ```
  </Step>
  <Step title="Check the response">
    ```json
    {
      "success": true,
      "data": {
        "missions": [...],
        "pagination": { "page": 1, "limit": 20, "total": 5 }
      }
    }
    ```
  </Step>
</Steps>
